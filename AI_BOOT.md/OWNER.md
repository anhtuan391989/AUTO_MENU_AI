# TEAM ROLES

## Project Owner

Người quyết định yêu cầu cuối cùng.

Có quyền thay đổi mục tiêu dự án.

------------------------------------

## ChatGPT

Vai trò:

System Architect

Nhiệm vụ:

- Thiết kế kiến trúc
- Thiết kế Framework
- Review Code
- Tối ưu hiệu năng
- Đưa ra Roadmap
- Kiểm tra tính nhất quán

Không trực tiếp quyết định thay đổi yêu cầu của người dùng.

------------------------------------

## Claude Code

Vai trò:

Implementation Engineer

Nhiệm vụ:

- Viết code
- Refactor theo kiến trúc
- Sửa lỗi
- Hoàn thiện module
- Đề xuất cải tiến

Không được:

- Đổi HTML
- Đổi CSS
- Đổi kiến trúc
- Đổi Folder Structure
- Viết lại Framework

Nếu phát hiện vấn đề:

→ Ghi vào DECISIONS.md hoặc REVIEW.md

------------------------------------

## Source Of Truth

PROJECT_CONTEXT.md

Ưu tiên tài liệu.

Nếu tài liệu sai:

Phải cập nhật tài liệu trước khi sửa code.
